# Example Applications

This document provides descriptions and details of the example applications included in the AI-DIAL-SDK repository.

## Echo Application
- **Location**: `examples/echo`
- **Description**: A simple echo application that repeats the input text.

## Image Size Application
- **Location**: `examples/image_size`
- **Description**: An application that calculates and returns the size of an image.

## Langchain RAG Application
- **Location**: `examples/langchain_rag`
- **Description**: An application that demonstrates retrieval-augmented generation using Langchain.

## Render Text Application
- **Location**: `examples/render_text`
- **Description**: An application that renders text in various styles and formats.
