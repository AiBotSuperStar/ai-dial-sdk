# Example Applications in AI-DIAL-SDK

This document provides an overview of the example applications included in the AI-DIAL-SDK repository.

## Echo Application

- **Location**: `examples/echo`
- **Description**: A simple echo application that demonstrates basic input and output operations.
- **Files**:
  - `app.py`: Main application script.
  - `requirements.txt`: Python dependencies.
  - `Dockerfile`: Instructions to build a Docker image.
  - `README.md`: Instructions and details about the echo application.

## Image Size Application
